Base <Base>

O base serve para facilitar na hora de acessar arquivos. Ele cria um repositório padrão para todo o site, por exemplo: Caso você tenha uma pasta com várias fotos chamada "assets" e use o base para dizer que o repositório padrão é essa pasta. Todos os links terão essa pasta como início. Exemplo: 

<base href="./assets/">

Quando você usa o base, todos os href irão começar a partir daquele link que for colocado no base. Por exemplo, caso você coloque "href = ./assets", qualquer href usado irá começar a partir de ./assets.

Só é possível usar uma tag base por página html.
